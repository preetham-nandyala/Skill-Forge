# Skill Forge
SkillForge is a web-based learning and assessment solution designed for aptitude preparation and coding practice.  

Key Features  
Secure authentication and role-based access control.  
Timed assessments with structured evaluation workflows.  
Real-time assessment event handling using Socket.IO.  

Tech Stack  
Frontend: Next.js, React.js  
Backend: Node.js, Express.js  
Database: MongoDB  
Security: JWT, RBAC  
Real-time: Socket.IO
